# ChatBot
"Este proyecto consiste en el desarrollo de una Aplicación Web (SPA) de chat colaborativo en tiempo real orientada a entornos corporativos. El sistema utiliza el protocolo WebSocket para establecer una comunicación bidireccional y simultánea entre múltiples usuarios, garantizando el intercambio de mensajes instantáneos sin técnicas de polling tradicional. La arquitectura cuenta con una API interna ligera para la gestión de conexiones y retransmisión de eventos, apoyándose en un Servicio de Identidad (IAM) para la autenticación de empleados o la asignación de nombres temporales, y una base de datos local para la persistencia del historial. Todo el desarrollo prioriza la simplicidad, el código limpio y la fácil ejecución."
