# ChatBot
Aplicación web colaborativa de mensajería en tiempo real que utiliza el protocolo WebSocket para establecer comunicación bidireccional persistente entre clientes y servidor. El sistema permite la interacción simultánea de usuarios, administración dinámica de conexiones y almacenamiento básico de información relevante.

## Configuración del Entorno de Desarrollo

Para levantar el servidor base y conectar con la base de datos de Supabase, sigue estos pasos:

### 1. Instalación de Dependencias
Ejecuta el siguiente comando en la terminal para instalar todas las librerías necesarias (Express, WebSockets, Supabase SDK y Dotenv):

npm install

### 2. Crear el archivo .env
creas el archivo fuera de public y colocas las credenciales que estan en el grupo de whattsapp

## Actualmente, el servidor no está encendido todo el tiempo. Si necesitan usar ejecuten el comando:

node server.js.
